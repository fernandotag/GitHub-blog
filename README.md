## How to run
**1.** Clone this project
```
git clone git@github.com/fernandotag/github-blog
``` 
**2.** Open the project folder github-blog in your system's shell
```
cd github-blog
``` 
**3.** install the project's dependencies
```
npm install
```
**4.** run the app
```
npm run dev
```
